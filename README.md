# 观天 PWA 原型

一个面向 iPhone 主屏幕使用的摄影天气 PWA，用来查看今日朝霞、晚霞、火烧云、星空概率和蓝调时间。

## 本地预览

在本目录的上一级运行：

```powershell
python -m http.server 5173 --bind 127.0.0.1 -d photo-weather-pwa
```

然后打开：

```text
http://127.0.0.1:5173/
```

## 当前功能

- 首页展示朝霞、晚霞、火烧云、星空概率。
- 顶部大图会根据晴天、多云、降雨、火烧云、夜间星空动态切换视觉状态。
- 点击顶部大图可打开今日总览详情。
- 点击朝霞、晚霞、火烧云、星空卡片可打开对应详情。
- 详情里展示地图、关键指标、判断依据、明天概率和附近候选方向。
- 星空详情提供光污染地图入口，并展示 CAMS 空气质量/气溶胶指标作为通透度参考。

## 数据和算法

- 天气数据来自 Open-Meteo 免费 Forecast API。
- 空气质量、颗粒物、气溶胶和沙尘指标来自 Open-Meteo Air Quality API，作为 CAMS 数据的浏览器友好入口。
- 定位使用浏览器 Geolocation。
- 朝霞/晚霞概率由中高云、低云、降水概率和能见度计算。
- 火烧云概率由中高云、低云、湿度、降水、能见度、气溶胶和 PM2.5 计算。
- 星空概率由总云量、降水概率、能见度、月光、PM2.5 和气溶胶估算。
- 蓝调时间用太阳高度角约 -4 到 -6 度估算。
- 附近候选点通过当前位置周边 8 个方向的免费天气预报粗略排序。

## iPhone 使用

部署到 HTTPS 后，用 iPhone Safari 打开网址，选择“分享”里的“添加到主屏幕”。
